- 文法
    
    - ==一、原因==
        
        |   |   |
        |---|---|
        |~ばかりに|正因為|
        |~だけに|正因為|
        |~だけあって|不愧|
        |~によって|因為|
        |~あまり|因過度…而|
        |~せいで|因…所害|
        |~おかげで|拜…所賜|
        
    
    - ==二、逆接==
        
        |   |   |
        |---|---|
        |1. ==ても類==||
        |~としても|倘若如此|
        |~名詞だって|就算 即使|
        |~たところで|就算 即使|
        |2. ==確定類==||
        |~ながらも|儘管…但|
        |3. ==いう類==||
        |~といっても|雖然…但|
        |4. ==のに類==||
        |~ くせに|卻|
        
    
    - ==三、累加==
        
        |   |   |
        |---|---|
        |~うえに|加上|
        |~に加えて|再加上|
        
    
    - ==四、手段==
        
        |   |   |
        |---|---|
        |~ によって|用 籍由|
        |~を通して|透過 藉由|
        |~を通じて|透過 籍由|
        |~をもって|以、用|
        
    
    - ==五、ところ==
        
        |   |   |
        |---|---|
        |~ところに~|在…時|
        |~たところ~|…之後|
        
    
    - ==六、根拠==
        
        |   |   |
        |---|---|
        |~によれば|根據 ..|
        |~に応じて|根據 .. 因應 ..|
        |~名詞では|根據 ..|
        |~をもとにして|根據 ..|
        
    
    - ==七、対象==
        
        |   |   |
        |---|---|
        |~にたいして|對於 .. 對待..|
        |~について|針對 .. 關於 ..|
        |~にこたえて|因應 .. 回應 ..|
        |~にかんして|關於..|
        |~にとって|對…而言|
        
    
    - ==八、部分否定==
        
        |   |   |
        |---|---|
        |~とはかぎらない|不一定|
        |~というわけではない|並非…|
        |~わけではない|並不是…|
        
    
    - ==九、変化==
        
        |   |   |
        |---|---|
        |~ほど~|越…越…|
        |~一方だ|越來越…|
        |~ばかりだ|越來越…|
        |~につれて|隨著…|
        |~にしたがって|隨著…|
        |~にともなって|伴隨…|
        |~とともに|…同時也…|
        
    
    - ==十、非限定==
        
        |   |   |
        |---|---|
        |~ばかりでなく|不光是|
        |~だけでなく|不只是|
        |~ のみならず|不只是|
        
    
    - ==十一、決心 決意==
        
        |   |   |
        |---|---|
        |~からには|既然…|
        |~た上は|既然…|
        |~た以上は|既然…|

        |   |   |
        |---|---|
