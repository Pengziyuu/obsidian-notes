# 四大詞性連接

## 四大詞性時態變化

### 丁寧體（敬語）

| 句型 | 時態 | [[動詞活用變化\|動詞]] | [[形容詞變化\|い形容詞]] | [[形容詞變化\|な形容詞]]・名詞 |
| --- | --- | --- | --- | --- |
| 肯定 | 現在 | 〇〇ます | 〇〇です | 〇〇です |
| 否定 |  | ません | くないです / くありません | では（じゃ）ありません / では（じゃ）ないです |
| 肯定 | 過去 | ました | かったです | でした |
| 否定 |  | ませんでした | くなかったです / くありませんでした | では（じゃ）ありませんでした / では（じゃ）なかったです |

### 普通體（口語、原形）

| 句型 | 時態 | [[動詞活用變化\|動詞]] | [[形容詞變化\|い形容詞]] | [[形容詞變化\|な形容詞]]・名詞 |
| --- | --- | --- | --- | --- |
| 肯定 | 現在 | 辭書形 | おいしい | 学生 だ |
| 否定 |  | ない形 | おいしくない | 学生 では（じゃ）ない |
| 肯定 | 過去 | た形 | おいしかった | 学生 だった |
| 否定 |  | なかった | おいしくなかった | 学生 では（じゃ）なかった |

## 詞性間的連接規則

### [[形容詞變化|い形容詞]]的連接

|  | + [[動詞活用變化\|動詞]] | + い形/な形 | + 名詞 |
| --- | --- | --- | --- |
| **い形** | ~~去い~~ + **く** | ~~去い~~ + **くて** | **直接連接** |
| 範例 | (新しい) 新し**く**始めます | (安い) この本は安**くて**便利です。 | **高い**ビルがあります。 |
| 範例 | (安い) 安**く**買いました。 | (面白い) その映画は面白**くて**人気です。 | **小さい**犬が好きです。 |

### [[形容詞變化|な形容詞]]的連接

|  | + [[動詞活用變化\|動詞]] | + い形/な形 | + 名詞 |
| --- | --- | --- | --- |
| **な形** | 以 **に** 連接 | 以 **で** 連接 | 以 **な** 連接 |
| 範例 | 静か**に**話します。 | 有名**で**明るい人です。 | 元気**な**子どもです。 |
| 範例 | 上手**に**歌います。 | 元気**で**にぎやか**な**学生です。 | きれい**な**花ですね。 |

### 名詞的連接

|  | + [[動詞活用變化\|動詞]] | + い形/な形 | + 名詞 |
| --- | --- | --- | --- |
| **名詞** | 以 **[[助詞]]** 連接 | 以 **が** 連接 | 以 **の** 連接 |
| 範例 | りんご**を**食べます。 | この本**が**安いです。 | 母**の**ごはんはおいしいです。 |
| 範例 | 日本語**を**勉強します。 | その先生**が**親切です。 | 学校**の**前にいます。 |

### [[動詞活用變化|動詞]]的連接

|  | + [[動詞活用變化\|動詞]] | + い形/な形 | + 名詞 |
| --- | --- | --- | --- |
| **動詞** | 用 **[[て形用法\|て形]]** 文法 | X | 動詞改為 **原形**（翻譯加「的」） |
| 範例 | (開ける) ドアを開け**て**入ります。 | X | **読む**本はこれです。 |
| 範例 | (行く) 行っ**て**買います。 | X | **走る**人がいます。 |
