## 擬聲語、擬態語

### 形式分類

1. **ABAB** 型
2. **AんBり** 型
3. **AっBり** 型
4. **~っ** 型

---

### ABAB 型

- **ふらふら** — 渾身沒力氣 / 漫無目的行動、迷糊 / 游移不定、猶豫
- **ぶらぶら** — 下垂搖晃 / 無所事事 / 悠閒自在到處走
- **からから** — 沒有水分乾透的狀態 / 器物裡什麼也沒有的狀態
- **がらがら** — 物體坍塌聲 / 開關拉門聲、車輪旋轉聲 / 漱口聲 / 聲音沙啞 / 人少
- **ぺらぺら** — 外語說得流利 / 連續翻開 / 輕易地說話 / 紙、布單薄
- **うろうろ** — 徘迴、轉來轉去 / 不知所措、驚慌失措
- **どきどき** — 心跳很快
- **はらはら** — 靜靜地飄落 / 擔心、憂慮、不安
- **さらさら** — 乾燥清爽、不黏稠
- **すらすら** — 流暢、輕鬆
- **ばらばら** — 零亂、零散

### AっBり 型

- **うっかり** — 不注意、不留神
- **がっかり** — 失望、灰心
- **しっかり** — 好好地 / 堅固 / 身心健全
- **すっかり** — 完全、徹底
- **すっきり** — 整潔 / 舒暢、清爽（= さっぱり）
- **はっきり** — 清楚、清晰 / 舒暢、清爽（= すっきり）
- **そっくり** — 一模一樣 / 全部
- **ぐっすり** — 熟睡、甜睡
- **さっぱり** — 直爽、坦率 / 味道清淡 / 完全、一點也不 / 舒暢、清爽（= すっきり）
- **ぴったり** — 緊密地 / 完全 / 恰好 / 說中、猜中
