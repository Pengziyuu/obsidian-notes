# 形容詞變化

## い形容詞與な形容詞的區別

- **い形容詞**：字尾==是「い」==。
- **な形容詞**：字尾==通常不是「い」==。

---

## い形容詞（形容詞）常見列表

- 高い (たかい) — 高的、貴的
- 低い (ひくい) — 低的
- 大きい (おおきい) — 大的
- 小さい (ちいさい) — 小的
- 新しい (あたらしい) — 新的
- 古い (ふるい) — 舊的
- 難しい (むずかしい) — 難的
- 易しい (やさしい) — 簡單的（內容）
- 暑い (あつい) — 熱的（天氣）
- 寒い (さむい) — 冷的（天氣）

## な形容詞（形容動詞）常見列表

- 静か (しずか) — 安靜的
- 賑やか (にぎやか) — 熱鬧的
- 簡単 (かんたん) — 簡單的（做法）
- 大変 (たいへん) — 辛苦的、嚴重的
- 親切 (しんせつ) — 親切的
- 元気 (げんき) — 有精神的、健康的
- 暇 (ひま) — 空閒的
- 丈夫 (じょうぶ) — 堅固的、健康的
- 不便 (ふべん) — 不方便的
- 便利 (べんり) — 方便的

> [!warning] な形容詞特例
> 以下字尾看起來像「い」，但其實是 **な形容詞**：
> - 綺麗 (きれい) — 漂亮的、乾淨的 *（字尾發音 rei）*
> - 有名 (ゆうめい) — 有名的
> - (大) 嫌い ((だい) きらい) — (最) 討厭的

## い形容詞活用變化

| 活用形 | 接續 | 以「暑い」為例 | 中文意思 |
| --- | --- | --- | --- |
| **未然形** | かろ + う | 暑かろう | 很熱吧 |
| **連用形** | かっ + た | 暑かった | 很熱（過去） |
|  | く + ない | 暑くない | 不熱 |
|  | く + て | 暑くて | 熱（中止形） |
| **終止形** | い | 暑い | （言畢） |
| **連體形** | い + とき | 暑いとき | 很熱的時候 |
| **假定形** | けれ + ば | 暑ければ | 如果很熱的話 |

### い形容詞特殊變化：「良い」（よい／いい）

| 活用形 | よい接續 | よい變化 | いい接續 | いい變化 | 中文意思 |
| --- | --- | --- | --- | --- | --- |
| **未然形** | よかろ + う | よかろう |  |  | 好吧 |
| **連用形** | よかっ + た | よかった |  |  | 太好了 |
|  | よく + ない | よくない |  |  | 不好 |
|  | よく + て | よくて |  |  | 好（中止形） |
| **終止形** | よい | よい | いい | いい | （言畢） |
| **連體形** | よい + とき | よいとき | いい + とき | いいとき | 很好的時候 |
| **假定形** | よけれ + ば | よければ |  |  | 如果可以的話 |

## な形容詞活用變化

| 活用形 | 接續 | 以「好き」為例 | 中文意思 |
| --- | --- | --- | --- |
| **未然形** | だろ + う | 好きだろう | 喜歡吧 |
| **連用形** | だっ + た | 好きだった | 曾經喜歡（過去） |
|  | で | 好きで | 喜歡（中止形） |
|  | に + なる | 好きになる | 變得喜歡（喜歡上了） |
| **終止形** | だ | 好きだ | （言畢） |
| **連體形** | な + とき | 好きなとき | 喜歡的時候 |
| **假定形** | なら +（ば） | 好きならば | 如果喜歡的話 |
