# 動詞種類

## 動詞屬性

| 動詞屬性 | 說明 | 例子 |
| --- | --- | --- |
| 狀態動詞 | 動詞本身==已是狀態==，不使用「ている」形式。 | ある、いる……等 |
| 繼續動詞 | 動作發生的時間==能持續==。 | 聞く、読む、書く、食べる、勉強する……等 |
| 瞬間動詞 | 動作發生的時間==很短暫==。 | 死ぬ、倒れる、壊れる、消える、結婚する……等 |
| 形狀動詞 | ==沒有時間對應==，宛如[[形容詞變化\|形容詞]]。在述部時必須使用「ている」形式。又稱第四種動詞。 | 優れる、似る、曲がる……等 |

## 自動詞 vs 他動詞

| 類別 | 說明 |
| --- | --- |
| **自動詞** | ==自發性==的動作或==自然產生==的動作、動作==不觸及他物==的動詞 |
|  | [[助詞]]：表主詞的「が、に」 |
|  | 走、來、有、在……等 |
| **他動詞** | 人或動物==對某物執行==的動作、動作==觸及他物==的動詞 |
|  | [[助詞]]：動作作用之對象的「を」 |
|  | 吃（飯）、喝（水）、讀（書）、寫（字）、穿（衣服）……等 |

> [!example] 自他動詞對照
> | 自動詞 | 他動詞 |
> | --- | --- |
> | ドア**が**開きます。（門開了。） | ドア**を**開けます。（開門。） |
> | 車**が**止まります。（車停了。） | 車**を**止めます。（停車。） |
> | 電話**が**壊れます。（電話壞了。） | 電話**を**壊します。（弄壞電話。） |
> | 紙**が**破れます。（紙破了。） | 紙**を**破ります。（撕紙。） |

> [!tip] 判斷技巧
> **「す」結尾的動詞必為「他動詞」**

> [!example] 「冷」的自他動詞體系
> | | 動詞 | 讀音 | 類別 |
> |---|---|---|---|
> | **高溫** ↓ | | | |
> | ① | 冷める | さめる | **自動詞** |
> | ② | 冷ます | さます | **他動詞** |
> | **常溫** ↓ | | | |
> | ③ | 冷える | ひえる | **自動詞** |
> | ④ | 冷やす | ひやす | **他動詞** |
> | **低溫** | | | |
> | ⑤ | 冷やかす | ひやかす | **他動詞** |

## 意志動詞 vs 非意志動詞

- **意志動詞**：動作者可以出於自己的意志去做的動詞。
  - 例：食べる、行く、書く、話す
- **非意志動詞**：動作者無法控制的、自然發生的動詞。
  - 例：降る、咲く、分かる

> [!info] 與[[使役形用法|使役形]]的關係
> 「意志動詞之自動詞」且「X 和 Y 皆為生命」時，使役形中「を」或「に」皆可使用。
