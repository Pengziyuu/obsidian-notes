## 敬語的五大分類

日語敬語自 2007 年文化審議會公告後，正式分為**五類**：

```mermaid
graph TD
    K[敬語] --> S[素材敬語<br/>針對話題中的人物]
    K --> T[對者敬語<br/>針對聽話的對象]
    S --> S1["① 尊敬語<br/>抬高對方"]
    S --> S2["② 謙讓語I<br/>降低自己(有對象)"]
    T --> T1["③ 謙讓語II/丁重語<br/>降低自己(無特定對象)"]
    T --> T2["④ 丁寧語<br/>禮貌表達"]
    T --> T3["⑤ 美化語<br/>美化用詞"]

    style S1 fill:#e8f5e9
    style S2 fill:#fff3e0
    style T1 fill:#fff3e0
    style T2 fill:#e3f2fd
    style T3 fill:#fce4ec
```

| 分類 | 功能 | 對象 | 代表例 |
|------|------|------|--------|
| **① 尊敬語** | ==抬高對方==的動作 | 上位者的動作 | いらっしゃる、お～になる |
| **② 謙讓語I** | ==降低自己==（有對象） | 自己→上位者的動作 | 伺う、お～する |
| **③ 謙讓語II（丁重語）** | ==降低自己==（無特定對象） | 自己的動作（禮貌描述） | まいる、おる、申す |
| **④ 丁寧語** | 禮貌表達 | 對聽者的禮貌 | です、ます |
| **⑤ 美化語** | 美化用詞 | 讓說法更優雅 | お料理、お花、ご飯 |

> [!warning] N2 關鍵區別
> **謙讓語I** 和 **謙讓語II（丁重語）** 的區分是 N2 最常考的重點！
> 詳見 → [[謙讓語#謙讓語I vs 謙讓語II（丁重語）]]

## 何時用哪一種？

```mermaid
flowchart TD
    A[要描述誰的動作？] -->|對方/上位者的動作| B[① 尊敬語]
    A -->|自己的動作| C{動作是否指向上位者？}
    C -->|是，有明確對象| D[② 謙讓語I]
    C -->|否，只是禮貌描述| E["③ 謙讓語II/丁重語"]
    A -->|只是想禮貌說話| F["④ 丁寧語(です/ます)"]
    A -->|想讓用詞更好聽| G["⑤ 美化語(お/ご)"]

    style B fill:#e8f5e9
    style D fill:#fff3e0
    style E fill:#fff3e0
    style F fill:#e3f2fd
    style G fill:#fce4ec
```

> [!tip] 記憶口訣
> - **尊敬語** = ==抬他==（把對方抬高）
> - **謙讓語I** = ==降己·有對象==（在 VIP 面前壓低自己）
> - **丁重語** = ==降己·無對象==（穿正裝說話，不管對誰）
> - **丁寧語** = ==禮貌==（基本的です/ます）
> - **美化語** = ==好聽==（加上お/ご讓詞彙優雅）
